Position CSS

  1. Static
  2. Relative
  3. Absolute
  4. Fixed
  5. Sticky

Static Position adalah posisi bawaan untuk semua elemen web. Jika tidak menentukan jenis posisi apapun, elemen tersebut akan memiliki static position secara default.

Relative Position adalah menetapkan elemen dengan relative position. Ini dapat menggerakan dari posisi asli tanpa mengganggu elemen lain di sekitarnya.

Absolute Position adalah berarti mengambil elemen keluar dari normal flow dan menetapkannya tepat di lokasi yang di inginkan.

Fixed Position adalah elemen dengan fixed position akan tetap berada di posisi yang sama, bahkan ketika audiens mengscroll halaman.

Sticky Position adalah kombinasi dari posisi relative dan fixed. Elemen akan bergerak seperti biasa sampai mencapai titik tertentu, lalu akan "menempel" (stick) seperti fixed position.
